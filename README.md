# Service_Tivi - Dịch Vụ Sửa Chữa Tivi Tại Cần Thơ

Website giới thiệu dịch vụ sửa chữa tivi chuyên nghiệp tại Cần Thơ.

## Công nghệ sử dụng

- React + Vite
- CSS3 với animations
- Responsive design

## Tính năng

- Giới thiệu dịch vụ
- Bảng giá dịch vụ
- Form đặt lịch hẹn
- Thư viện hình ảnh
- Phản hồi khách hàng
- Thông tin liên hệ
